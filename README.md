## 🏗️ Master Course Architecture
The course is divided into 5 Chapters, each focusing on a specific SDK ecosystem and a corresponding **Business Case Study.**

*   **Business Case**: 5-Agent Financial Boardroom (MSFT/TSLA Q3 Report) & Brand Compliance Plugin Pipeline.

*   **Business Case**: 7-Section HR Knowledge Agent & 7-Node Intelligent Support Router with VIP Escalation.

### [Chapter 3: Native Provider Tooling](./Chapter_3_OpenAI_and_Google.md)
*   **Tools**: OpenAI Assistants API, Google Gen AI SDK (ADK).
*   **Business Case**: 3-Clause Legal Contract Auditor & 3-Market PropTech Investment Analysis.

### [Chapter 4: Specialized Open-Source](./Chapter_4_Specialized_Open_Source_Frameworks.md)
*   **Tools**: CrewAI, HuggingFace smolagents.
*   **Business Case**: 4-Agent Marketing Swarm with Brand Compliance & Multi-Region Tax Discrepancy Auditor.

*   **Tools**: Dify, n8n.
*   **Business Case**: Headless Sales Concierge (REST API) & VIP Incident Escalation Swarm (Webhook Actions).

## ⚡ Interactive Learning Environment
Every chapter includes a **Premium Interactive Notebook** that you can run locally:
*   **Visual UI Contexts**: Every agent demo features a custom-designed, high-fidelity UI rendered directly in your notebook (Brutalist, Glassmorphism, and Dark Mode terminal aesthetics).
*   **Cost & Speed Optimized**: All notebooks use `gpt-4o-mini` by default (0.15¢ per 1M tokens) to ensure your learning is virtually free.
*   **Hybrid Simulation Mode**: No API Keys? No problem. Every notebook features a high-fidelity simulation mode so you can see the agentic behavior regardless of your environment.
*   **Silent Execution**: `%%capture` magic hides all framework logs — output is rendered via premium HTML/CSS UIs only.

## 🛠️ Getting Started
1. Clone this repository.
2. Rename `.env.example` to `.env` and add your keys (OpenAI, Google, etc.).
3. Run `pip install -r requirements.txt`.
4. Open any notebook in the `Code_Examples/` folder to begin!
